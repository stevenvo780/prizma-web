Humanizar Web (React)

Sitio corporativo de Humanizar migrado a React + Vite con enfoque comercial.

Stack
- React 18
- Vite 5
- CSS modular por componentes
- Deploy estatico (Vercel) y Docker + Nginx
- Iconografia local: `public/assets/icons/apps` y `public/assets/icons/clients`

Identidad empresarial publicada
- Mision: impulsar crecimiento empresarial con software conectado y util.
- Vision: ser plataforma confiable en Colombia para ventas, operacion y colaboracion.
- Clientes importantes: bloque con clientes destacados (sin exponer cifras internas ni datos confidenciales).
- Logos de clientes y apps descargados y servidos localmente desde el proyecto.
- Marco organizacional publicado: narrativa, arquitectura de portafolio, ICP, implementacion, modelo comercial, roadmap, operacion y cultura.

Documentos empresariales
- `ORGANIZACION_EMPRESA.md`: marco operativo y estrategico de HumanizarSystems.

Indicadores corporativos documentados (sin cifras sensibles)
- KPIs por software (anexo): `KPIS_SOFTWARE.md`.

Comandos
- Instalar: npm install
- Desarrollo: npm run dev
- Build: npm run build
- Preview: npm run preview
